# ♻️ Plastic Bottle Reward dApp (Recycle-to-Earn)

## Overview

This dApp allows users to record the number of plastic bottles they recycle and earn rewards.

Users can:

* Deposit the number of bottles collected
* Track their total contribution
* Redeem rewards (reset their balance)

## Concept

This project simulates a **sustainable recycling incentive system** using blockchain technology.

Each deposit represents recycled plastic bottles, and rewards can be claimed based on total contributions.

## Features

* Track recycled plastic per user
* Accumulate contributions
* Redeem rewards
* Simple and transparent system

## Use Case

Encourages environmental awareness by rewarding users for recycling plastic waste.
